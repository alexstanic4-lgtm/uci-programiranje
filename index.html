Uvod u Programiranje

Programiranje je pisanje instrukcija za računalo. Koristimo jezike kao JavaScript.

console.log("Zdravo, svijete!");

Varijable

Varijable spremaju podatke. Koristimo let ili const.

let ime = "Ana";
const godine = 16;
console.log(ime + " ima " + godine + " godina.");

Uvjeti (If-Else)

Uvjeti provjeravaju istinitost i izvršavaju kod.

let broj = 10;
if (broj > 5) {
    console.log("Broj je veći od 5");
} else {
    console.log("Broj je manji ili jednak 5");
}

Petlje

Petlje ponavljaju kod. For petlja za poznati broj ponavljanja.

for (let i = 0; i < 5; i++) {
    console.log("Broj: " + i);
}

Nizovi (Arrays)

Nizovi spremaju više vrijednosti.

let boje = ["crvena", "plava", "zelena"];
console.log(boje[0]); // crvena
boje.push("žuta");
console.log(boje.length); // 4

Funkcije

Funkcije su blokovi koda koji se mogu pozvati više puta.

function pozdrav(ime) {
    return "Zdravo, " + ime + "!";
}
console.log(pozdrav("Marko"));

Objekti

Objekti spremaju podatke u svojstvima.

let ucenik = {
    ime: "Ana",
    godine: 16,
    razred: "8.b"
};
console.log(ucenik.ime); // Ana
